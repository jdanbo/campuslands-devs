# Comandos

<strong> Comando `cd`:</strong>
Este comando es utilizado par aingresar a carpeta desde la terminal, en caso de querer salir de la carpeta se utiliza la expresion `cd ..`.

<strong> Comando `mkdir`:</strong> El comando `mkdir` es utilizado para la creacion de directorios.

<strong> Comando `gir status`:</strong> Es un comando escensial en el control de cambios dentro de un proyecto o repositorio.

<strong> Comando `git clone `:</strong> se utiliza para clonar un repositorio en nuestro editor de codigo.

<strong> Comando `git branch`:</strong> Este comando es utilizado para verificar en donde se esta ubicado, si se utiliza `git branch <nombre>` se creara una rama en nuetro repositorio.
