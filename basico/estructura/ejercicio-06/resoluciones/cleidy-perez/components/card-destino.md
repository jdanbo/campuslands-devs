# Componentes: Card Destino

## Descripción
Este componente visual se encarga de renderizar la información resumida de un destino turístico en formato de tarjeta.

## Propiedades
- `nombre` (String): Nombre del lugar turístico.
- `pais` (String): País de ubicación.
- `precio` (Number): Costo por noche en USD.

## Ejemplo de Uso
```html
<card-destino 
  nombre="Machu Picchu" 
  pais="Peru" 
  precio="150">
</card-destino>
