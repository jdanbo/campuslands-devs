# Ejercicio 04 - Estructura

## Análisis de Proyeecto
Se necesita crear una base para un proyecto relacionado con autos de luj. Organizar datos, imagenes y documentacion de catalogo premium. 

Requerimientos
    Crea data, assets/cars, docs y src.
    Incluir archivo ejemplo
    Conveciones para imagenes
    Logica en src
    checklist
