# Taller de Motos - Solución ejercicio 04 Estructura

## Henrik Anderson Oloroso García


## Arbol de archivos
``` text
taller_de_motos/
├─ config/
│  └─ settings.example.json
├─ docs/
│  └─ flujo-reparacion.md
└─ modules/
   ├─ clientes/
   ├─ motos/
   └─ ordenes/

```

_Justificación_: En base a las necesidades de el taller de motos se obtuvo la estructura de árbol anterior.
- Como carpeta principal está `taller_de_motos`
- `config`: En esta carpeta reside las configuraciones básicas para la organización.
- `docs`: Aqui reside la documentación del flujo de trabajo y su reparación
- `modules`: Aqui se organizan los datos de los clientes, las motos y sus ordenes.

En base a las instrucciones, pensé que esta sería la solución más óptima para el problema de estructura
