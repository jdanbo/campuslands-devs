# Backend Battle Royale

**Nombre del Camper:** Marco Antonio

## Descripción
Estructuración de carpetas para un Backend básico para gestionar equipos y partidas de un torneo battle royale.

## Solución

Estructura de carpetas creada para la simulación del proyecto Battle Royal:

```
antonio-canux/
├── src/
|   ├── controllers/
|   │   └── teamController.js
|   ├── services/
|   │   └── teamService.js
|   ├── models/
|   │   └── teamModel.js
|   └── routes/
|       └── teamRoute.js
└── antonio-canux.md
```

## Estructura

- **controllers**: Manejo de peticiones.
- **services**: Lógica de negocio.
- **models**: Definición de datos.
- **routes**: Definición de rutas.
