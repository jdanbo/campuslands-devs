# Ejercicio 0 - Lógica

## Análisis de Proyecto
Un club de autos de lujo quiere clasificar vehículos por aceleración y exclusividad. Filtrar autos por 0-100 km/h, precio y unidades producidas.

Requerimientos:
1. Define autos con marca, modelo, ceroACien, precioUSD y unidades.
2. Filtra los que hacen 0-100 en menos de 3 segundos.
3. Clasifica exclusividad: unidades < 500 es extrema.
4. Calcula precio promedio.
5. Muestra top 3 por aceleracion.
