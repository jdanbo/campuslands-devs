Ejercicio 02 - Lógica 
Entendimiento del problema

Equipo de shooters necesita un sistema para filtrar armas disponibles, validar su municion disponible y recomendar cargas para el combate. 

Requerimientos
    BD de armas con su informacion necesaria
        nombre, tipo, municion, rareza
    Filtro por cantidad de municion
    Tipos
        rifles, pistolas, francotiradores
    Notificar de rareza o epicas
    Imprimir resumen por tipo
