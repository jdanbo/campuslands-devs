# Ejercicio 07 - Lógica

## Análisis de Proyecto
Un mecánico recibe síntomas reportados por pilotos después de una carrera. Usar condicionales para sugerir posibles causas mecánicas.

Requerimientos:
1. Crea una funcion diagnosticar(sintomas).
2. Evalua sintomas como 'no enciende', 'vibra', 'pierde aceite', 'frena poco'.
3. Devuelve una recomendacion por sintoma.
4. Permite varios sintomas en un arreglo.
5. Genera un reporte final.
