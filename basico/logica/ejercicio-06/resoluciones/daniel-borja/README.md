# Ejercicio 06 - Lógica

## Análisis de Proyecto
Un taller compara motos por potencia, peso y costo de mantenimiento. Crear una métrica simple de rendimiento y clasificar las motos.

Requerimientos: 
1. Guarda motos con marca, modelo, hp, pesoKg y mantenimientoMensual.
2. Calcula relacion peso-potencia = hp / pesoKg.
3. Filtra motos con mantenimiento menor a 500000.
4. Ordena por mejor relacion.
5. Muestra la mejor opcion para pista y la mejor opcion económica.
