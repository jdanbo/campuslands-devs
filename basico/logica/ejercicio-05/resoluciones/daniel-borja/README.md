# Ejercicio 05 - Lógica

## Análisis de Proyecto
Una liga de futbol necesita actualizar una tabla luego de su partido. Se necesita: Calcular puntos, diferencia de goles y ordenar equipos.

Requerimientos:
1. Crea equipos con victorias, empates, derrotas, golesFavor y golesContra.
2. Calcula puntos = victorias * 3 + empates.
3. Calcula diferencia de goles.
4. Ordena por puntos y luego diferencia.
5. Imprime tabla con posicion.
